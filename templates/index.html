Documentation

Endpoints

HTTP Method Endpoint Description Requires API Key
GET / Returns the homepage. No
GET /documentation Returns the documentation. No
GET /inventory Returns a JSON object containing all products available for purchase. No
PATCH /update_inventory/<id> Updates the product with the specified ID. Yes
POST /add_inventory Adds a new product to the inventory. Yes
DELETE /delete_inventory/<id> Deletes the product with the specified ID. Yes

API Key

The API key is required for endpoints that modify the inventory. To use these endpoints, include the API key in the request headers with the key api_key.

Schema

The database schema for the inventory table is as follows:

Column Type Description
id integer Unique ID for the product.
name string Name of the product.
price float Price of the product in USD.
size string Size of the product.
quantity integer Quantity of the product available for purchase.
image_url string URL of the product image.
sku string Stock Keeping Unit (SKU) for the product.
